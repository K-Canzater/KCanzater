Architecture Overview

This project's architecture centers around a secure, internet-enabled VPC built entirely through AWS CLI automation. The infrastructure includes a custom VPC with CIDR block configuration, a public subnet strategically placed for internet-accessible resources, and an Internet Gateway enabling seamless external communication. Route tables are programmatically updated to direct traffic flow, while the entire network topology is designed for scalability and security. The CLI-driven approach demonstrates infrastructure-as-code principles, allowing for rapid deployment and consistent configuration across environments.
