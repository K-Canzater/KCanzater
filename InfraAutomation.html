Architecture Overview

The project’s architecture comprises a VPC with public and private subnets, an internet gateway, route tables, and security groups. An EC2 instance resides within the public subnet, with an S3 bucket configured for storage. IAM roles and policies manage permissions. GitHub Actions automate deployments, while Boto3 scripts manage AWS resources post-deployment.

Implementation Steps

  • Defined VPC, subnets, route tables, internet gateway, security groups, EC2, S3 bucket, and IAM roles using Terraform.
  • Utilized variables for flexible configuration and scalability.
  • 🎯 Key Impact: Achieved 100% automated infrastructure provisioning with zero manual configuration.
